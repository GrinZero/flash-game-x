## 📦 提交游戏

我们约定，一个游戏，应该包含这些东西：

- 游戏名称。暂时中英混合吧
- 游戏的唯一 key。该属性对应`public`目录下的文件夹名称
- 游戏的资源文件。参考`public`目录下的文件夹结构
- 游戏的封面图片。建议 w:h=1:1.1，即高度为宽度的 1.1 倍
